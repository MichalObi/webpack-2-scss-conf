### Install Webpack v.2

```sh
$ npm i /-g/ webpack@2 webpack-dev-server@2
```

### Install appropriate loader: 
- Handle css
    ```sh
    $ npm i css-loader
    ```
- Adding css to DOM - <style> tag injection
    ```sh
    $ npm i style-loader
    ```
- Handle scss
    ```sh
    $ npm i sass-loader node-sass
    ```
- Place css in separate fil
    ```sh
    $ npm i extract-text-webpack-plugin@2.0.0-beta.5
    ```
### Run Webpack
 ```sh
    webpack /-p/
```
